# Toadler

**Toadler** is a 2D arcade survival game developed in **Unity**. You play as a frog who must jump across lily pads to survive as the water level rises. The game tests your timing and precision as lily pads drown over time, requiring quick decisions to stay afloat.

### Setup & Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/rakeshmalik91/toadler.git
   ```
2. Open the project folder `toadler.unity6.3` in **Unity Hub**.
3. Load the main scene from `Assets/Scenes`.
4. Press **Play** to test in the editor.
